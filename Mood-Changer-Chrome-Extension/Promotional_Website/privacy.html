Privacy Policy

Last updated: April 23, 2026

EmoUI ("the Extension") is a Chrome extension that uses on-device artificial intelligence to detect your emotional state via your webcam and enhance your browsing experience. This privacy policy explains how EmoUI handles your data.

Summary: EmoUI processes all camera data locally on your device. We collect zero personal data. No images, video, or facial data ever leave your computer.

1. Information We Collect

EmoUI does not collect, store, or transmit any personal information. Specifically:

  • Camera Data: Your webcam feed is processed entirely on-device using ONNX Runtime Web (WebAssembly). Video frames are analyzed in memory for mood detection and are immediately discarded. No images, video frames, facial data, or biometric information are ever stored, transmitted, or logged.
  • Mood Data: Detected mood readings (e.g., "positive", "negative", "neutral") exist only in the extension's volatile runtime memory and are not persisted or transmitted.
  • Browsing Data: EmoUI does not access, monitor, or collect any browsing history, page content, URLs, or cookies.
  • Analytics: EmoUI does not use any analytics, telemetry, tracking scripts, or third-party monitoring services.

2. Information You Provide

The following information is optionally provided by you and stored locally:

  • OpenAI API Key: If you choose to enable AI-powered quotes, you may enter your OpenAI API key. This key is stored exclusively in your browser's local storage (chrome.storage.local) and is never shared with EmoUI's developers or any third party. The key is sent only to OpenAI's servers (api.openai.com) when generating motivational quotes.
  • Settings: Your preferences (detection interval, language, music preference) are stored locally in your browser using Chrome's storage API. These settings never leave your device.

3. Third-Party Services

EmoUI makes external network requests only in the following scenario:

  • OpenAI API (optional): If you have provided an OpenAI API key, the extension sends a short text prompt (e.g., "Generate a soothing quote for someone feeling sad") to the OpenAI Chat Completions API. This request includes your API key and the text prompt only. No camera data, personal information, facial data, or browsing data is included in this request. OpenAI's use of this data is governed by OpenAI's Privacy Policy.

If no API key is provided, EmoUI operates entirely offline and makes zero network requests.

4. Permissions Justification

EmoUI requests the following Chrome permissions:

  • offscreen: Required to create an offscreen document that accesses the webcam for mood detection. This runs in the background without interrupting your browsing experience.
  • notifications: Used to display mood-based notifications and soothing music suggestions through Chrome's native notification system.
  • storage: Used to persist your user preferences (detection interval, language, API key) locally within your browser.

5. Data Security

Since all AI processing occurs on-device and no personal data is collected or transmitted, there is no centralized data store to protect. Your OpenAI API key is stored in Chrome's sandboxed local storage, which is isolated from other extensions and websites.

6. Children's Privacy

EmoUI is not directed at children under the age of 13. We do not knowingly collect any personal information from children.

7. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. Continued use of the Extension after changes constitutes acceptance of the updated policy.
